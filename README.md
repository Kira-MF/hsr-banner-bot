# HSR Banner Bot

Telegram-бот для отслеживания баннеров Honkai: Star Rail.

## Что умеет

- показывает текущие баннеры персонажей;
- показывает баннеры световых конусов;
- даёт подписки на персонажей;
- уведомляет о новых баннерах;
- хранит пользователей и подписки в SQLite.

## Установка

```powershell
py -m pip install -r requirements.txt
```

## Настройка

Создай `.env`:

```env
TG_BOT_TOKEN=токен_telegram_бота
```

## Запуск

```powershell
py main.py
```

База `hsr_bot.db` создаётся автоматически при первом запуске.
